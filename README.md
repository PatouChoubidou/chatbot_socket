# Chatbot with socket
This is test for chatting with uploaded document-files locally.  
Handles pdf, docx, rtf, txt files

## Tools in use 
### backend
- Ollama 
- Fastapi
- Whisper
- Chroma DB 

### frontend
- Vue.js


## Usage
#### backend
source .venv/bin/activate
fastapi dev app.py

#### frontend
npm run dev
